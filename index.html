Open Mod Loader

A fully open source modern Minecraft mod loader

Open Mod Loader is a full replacement for other mod loaders, It is currenly in development and is not ready to be played with.

Find Out More

Fully open source

All of the tools that we use are made open source under the MIT license. They can be found on GitHub, feel free to make a pull request or fork the project!

Github

Features

Speed

The mod loader has been designed to reduce the overhead of running mods.

Updates

The mod loader is updated to new Minecraft versions as soon as possible.

API's

The mod loader has a wide range of API's for mod developers.

Easy

The mod loader is easy to use, documentation on how to use it will become available over time.

Download

There is not ETA on a download of this project, it will be ready once it's ready.
